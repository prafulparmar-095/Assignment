1. What is JavaScript Output method?
Ans => JavaScript output refers to the information, results, or content that a JS program displays for users. Output in JavaScript can be displayed in various forms, and developers often use different methods to display the output of a given code.
2. How to used JavaScript Output method?
Ans =>
  1. innerHTML - Display the HTML output on an element.
  2. Using console.log() Method - Display the content on the browser console.
  3. document.write() method - Displays the content on an HTML element.
  4. window.alert() Method - Display the HTML element to the output.
