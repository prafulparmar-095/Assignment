Redux Question Answer

1.What is Redux?
Ans = Redux is a predictable state container for JavaScript apps.
2.What is Redux Thunk used for?
Ans = Redux Thunk is used for Asynchronous actions, such as fetching data from an API or performing asynchronous computations, are common in web applications.
Redux-Thunk enables you to dispatch actions asynchronously, making it easier to manage side effects in your Redux applications.
3.What is Pure Component? When to use Pure Component over Component?
Ans =It is one of the most significant ways to optimize React applications. By using the pure component, there is no need for shouldComponentUpdate() Lifecycle Method as ReactJS
Pure Component Class compares current state and props with new props and states to decide whether the component should re-render or Not.
4.What is the second argument that can optionally be passed tosetState and what is its purpose?
Ans = The second parameter to setState() is an optional callback function that will be executed once setState is completed and the component is re-rendered.
